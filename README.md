# takingCareApp
Aplicación en java y mysql para registrar y rastrear hábitos personales: alimentación, vitaminas, estado físico y emocional, ejercicio y sueño. 
Incluye un sistema que recomienda comidas de forma aleatoria sin que estas se reptitan si una de las comidas ya fue tomada en la última semana.

Tecnologías
- Java
- MySQL
- JDBC

Estructura de la base de datos
Incluye las siguientes entidades: Cliente, Comidas, Comidas_Recomendacion, Vitaminas, EstadoEmocional, EstadoFisico, Sueno, EjercicioFisico.
